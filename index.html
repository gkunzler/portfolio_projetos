Projetos em Ciência de Dados

Projeto de predição de vendas Rossmann

Neste projeto foi realizada uma predição de vendas para as próximas seis semanas das lojas da Rossmann. O projeto passou por 11 etapas: Carregamento de dados; Limpeza e descrição dos dados; Criação de features; Análise exploratória dos dados; Preparação dos dados; Seleção de features com boruta; Comparação entre modelos de Machine Learning; Refinamento de parâmetros do modelo; Interpretação do erro; Performance total do modelo; Publicação do modelo em produção e criação de bot no telegram

Ferramentas utilizadas:

  • Python
  • Jupyter Notebook
  • Pandas, Matplotlib, Plotly
  • Borutapy, Sklearn - RobustScaler,MinMaxScaler, LabelEncoder, LinearRegression, Lasso
  • Github
  • Subsistema Windows para Linux - WSL2

Projeto de coleta de dados via Webscrpapin e análise de dados para uma empresa de e-commerce

Neste projeto foram coletados dados via Webscraping do site da H&M. A coleta de dados foi automatizada através de uma rotina, que capturou os dados do site 2 vezes ao dia, durante quatro dias. Os dados foram inseridos em um banco de dados, e analisou-se os produtos do setor de calças jeans masculinas. O objetivo do projeto era fornecer informações uteis para empresa Star Jeans sobre os seus concorrentes.

Ferramentas utilizadas:

  • Python
  • Jupyter Notebook
  • Beautiful Soup, Pandas, Seaborn, Matplotlib
  • SQLite

Projeto de Insights para determinar lista de imóveis a serem comprados por uma empresa do ramo imobiliário

O objetivo deste projeto foi identificar para uma empresa do ramo imobiliario uma lista de imoveis para compra, que renderiam bons lucros para empresa. O projeto passou por uma etapa de insights, em que foram analisados os dados dos imóveis.
Após isso foi gerado uma lista de imoveis para compra, e criado um sistema de precificação para determinar o preço de venda e lucro destes imoveis.
Além disso, foi desenvolvido um aplicativo para visualização dos dados.

Ferramentas utilizadas:

  • Python
  • Jupyter Notebook, Pycharm
  • Pandas, Seaborn, Matplotlib
